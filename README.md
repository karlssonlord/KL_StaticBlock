# KL_StaticBlock

## Installation

The easiest way to install the module is by using [modman](https://github.com/karlssonlord/modman):

1. `modman clone git://github.com/karlssonlord/KL_StaticBlock.git`
2. `modman deploy KL_StaticBlock`
